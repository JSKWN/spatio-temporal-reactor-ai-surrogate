# SE-A1 컴포넌트 기능 테스트 보고서 — `layers3d.py`

> **작성일**: 2026-04-08 (초판), **2026-04-15 재작성** (halo 6×6 격자 반영)
> **대상 파일**: `src/models/spatial/layers3d.py`
> **검증 단계**: SE-A1 (Spatial Encoder, Phase A, Step 1)

---

## 1. 목적

공간 인코더의 기본 layer 3종 — `CellEmbedder`, `FFN3D`, `LearnedAbsolutePE3D` — 의 단독 동작 검증.
(ConditionalLAPE3D는 SE-A3 보고서에서 별도 검증)

각 layer가:
- 의도한 텐서 형상 변환을 수행하는가
- 파라미터 수가 설계 의도와 일치하는가 (수식 검증)
- 기본 forward 동작이 오류 없이 실행되는가

## 3. 검증 항목 및 결과

### 3.1. 형상(Shape) 변환

| 컴포넌트 | 입력 형상 | 출력 형상 | 결과 |
|---|---|---|:---:|
| `CellEmbedder(d_latent=128)` | `(2, 20, 6, 6, 21)` | `(2, 20, 6, 6, 128)` | PASS |
| `LearnedAbsolutePE3D(20, 6, 6, 128)` | `(2, 20, 6, 6, 128)` | `(2, 20, 6, 6, 128)` | PASS |
| `FFN3D(d_latent=128, expand=4)` | `(2, 720, 128)` | `(2, 720, 128)` | PASS |
| `LearnedAbsolutePE3D.get_norm_map()` | — | `(20, 6, 6)`, `float32` | PASS |

### 3.2. 파라미터 수 — 수식 검증

| 컴포넌트 | 측정값 | 수식 | 기대값 | 결과 |
|---|---:|---|---:|:---:|
| `CellEmbedder` | **2,816** | `21·128 + 128` (Conv3D 1×1×1 weight + bias) | 2,816 | PASS |
| `LearnedAbsolutePE3D` | **92,160** | `20·6·6·128` (단일 trainable variable) | 92,160 | PASS |
| `FFN3D (expand=4)` | **131,712** | `(128·512 + 512) + (512·128 + 128)` | 131,712 | PASS |

### 3.3. `LearnedAbsolutePE3D` 분석 hook 동작

| 항목 | 값 | 결과 |
|---|---|:---:|
| `get_norm_map()` 반환 형상 | `(20, 6, 6)` | PASS |
| `get_norm_map()` 반환 dtype | `float32` | PASS |

## 4. 비교 박스 — 설계 의도 vs 측정

```
┌──────────────────────────────────────────────────────────────┐
│  CellEmbedder (Conv3D 1×1×1, 21 → 128)                      │

│  설계 의도: per-cell channel projection, spatial mixing 0   │
│  측정 결과:                                                   │
│    - shape (2,20,6,6,21) → (2,20,6,6,128)  PASS             │
│    - params 2,816 = 21·128 + 128            PASS             │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│  LearnedAbsolutePE3D ((20,6,6,128) trainable embedding)     │
├──────────────────────────────────────────────────────────────┤
│  설계 의도: 720 cell × 128차원 정체성 vector, BC 식별 자유   │
│  측정 결과:                                                   │
│    - shape (2,20,6,6,128) → (2,20,6,6,128)  PASS            │
│    - params 92,160 = 20·6·6·128             PASS             │

└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│  FFN3D (Dense expand×4, GELU)                                │
├──────────────────────────────────────────────────────────────┤
│  설계 의도: token-wise 비선형 (attention 후 정보 처리)         │
│  측정 결과:                                                   │
│    - shape (2,720,128) → (2,720,128)        PASS             │

└──────────────────────────────────────────────────────────────┘
```
