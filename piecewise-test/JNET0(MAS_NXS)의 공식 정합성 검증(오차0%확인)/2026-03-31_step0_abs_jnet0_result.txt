1. 검증 목적
============================================================
MAS_NXS 파일에서 출력되는 ABS, JNET0 컬럼의 물리적 정의를 실제 데이터로
역산하여 확정. 6가지 가설 조합을 시도하고, 밸런스 잔차가 최소인 조합 채택.

============================================================
2. 검증 방법: 단일 노드 중성자 밸런스 잔차
============================================================
MASTER가 출력한 노드별 물리량(MAS_NXS)을 이용해, 이론적으로 0이 되어야 하는
중성자 밸런스 방정식의 잔차를 계산. 잔차가 작을수록 올바른 해석.

■ 2군 확산 밸런스 방정식 (체적 적분형):

  ┌─ g=1 (고속군, χ₁ = 1) ─────────────────────────────────┐
  │                                                          │
  │  Leakage_g1 + Removal_g1 = FissionSource                │
  │                                                          │
  │  JNET_sum_g1 + (ABS_g1 + SCA) × φ₁ × V                 │
  │    = (1/keff) × (NFS_g1 × φ₁ + NFS_g2 × φ₂) × V       │
  │                                                          │
  │  좌변: 누설 + 제거 (흡수 + 산란out)                      │
  │  우변: 핵분열원 전체 (χ₁=1이므로 g1에 모두 배정)         │
  └──────────────────────────────────────────────────────────┘

  ┌─ g=2 (열중성자군, χ₂ = 0) ──────────────────────────────┐
  │                                                          │
  │  Leakage_g2 + Removal_g2 = ScatterSource                │
  │                                                          │
  │  JNET_sum_g2 + ABS_g2 × φ₂ × V = SCA × φ₁ × V         │
  │                                                          │
  │  좌변: 누설 + 흡수 (열군에서는 scatter-out 없음)         │
  │  우변: g1→g2 산란 전입원 (핵분열원 없음, χ₂=0)          │
  └──────────────────────────────────────────────────────────┘

■ 핵분열 스펙트럼 χ_g:
  핵분열 중성자는 MeV 에너지로 태어나므로 고속군에 전부 배정.
    χ_g1 = 1 (고속군): 핵분열원 전체 수신
    χ_g2 = 0 (열중성자군): 핵분열원 없음, scatter-in만

■ 각 항의 물리량 출처 (MAS_NXS 컬럼):
  ABS_g    = ABS 컬럼 (Σ_a = Σ_c + Σ_f, 흡수 거시단면적 [/cm])
  SCA      = SCA 컬럼 (Σ_s12, g1→g2 산란 거시단면적 [/cm], g2에서는 0)
  NFS_g    = NFS 컬럼 (νΣ_f [/cm])
  φ_g      = FLX 컬럼 (노드 평균 중성자속 [/cm²/s])
  V        = dx × dy × dz (노드 체적 [cm³])

■ JNET0 누설항 계산:
  JNET0은 per-unit-area [n/cm²/s], positive = 노드 외부 방향 (leakage 양수).
  각 면의 누설 기여 = JNET0_face × A_face:
    N/S 면: A_ns = dx × dz  (X-Z 면)
    W/E 면: A_we = dy × dz  (Y-Z 면)
    B/T 면: A_bt = dx × dy  (X-Y 면)
  JNET_sum = JNET0_N×A_ns + JNET0_S×A_ns
           + JNET0_W×A_we + JNET0_E×A_we
           + JNET0_B×A_bt + JNET0_T×A_bt   [n/s]

■ 잔차 정의:
  R_g = |Leakage + Removal - Source|
  상대잔차(%) = R_g / |Removal_g| × 100

  Removal_g = (ABS_g + SCA_g) × φ_g × V  (g=1)
            = ABS_g × φ_g × V             (g=2)
  → 반응률 대비 밸런스 미달 정도를 측정.

6. 확정 결과 (최종 — JNET0 스케일 팩터 발견 후 수정)
============================================================
┌──────────────────────────────────────────────────────────┐
│ 1. ABS = Σ_a = Σ_c + Σ_f (총 흡수 거시단면적 [/cm])     │
│ 2. JNET0 = half net current (net current의 절반으로 출력) │
│    per-unit-area [n/cm²/s], positive = outward           │
│ 3. 밸런스 시 ×2 × 면적 곱셈 필수:                        │
│    leak = 2 × Σ(JNET0_face × A_face)                    │
└──────────────────────────────────────────────────────────┘

■ JNET0 스케일 팩터 발견:
  α=1 사용 시 g1 median 3.19% 잔차 발생 (Step 0 초기 결과).
  최적 α 탐색 결과: median α = 2.0000, std = 0.03.
  α=2 적용 시 g1/g2 median 0.0002% → 밸런스 완벽 성립.

  초기 관찰된 Z축/XY 공간 패턴 (축방향 상·하부 잔차 급증,
  XY 체커보드)은 모두 α=1 스케일 오류의 아티팩트였음.
  α=2 적용 후 모든 공간 패턴 소멸, 전 영역 균일하게 ~0%.

  → MAS_NXS 물리량(ABS, SCA, NFS, FLX, JNET0×2)으로
    MASTER 노드 밸런스를 **정확히** 재현 가능.
