N/S 매핑 최종 확정
============================================================
확정된 JNET0 방향 매핑 (I,J = MAS_NXS 1-based):
  E = +I 방향: (I,J).E + (I+1,J).W = 0  ✓
  W = -I 방향: (I,J).W + (I-1,J).E = 0  ✓
  S = +J 방향: (I,J).S + (I,J+1).N = 0  ✓
  N = -J 방향: (I,J).N + (I,J-1).S = 0  ✓

핵심: J 인덱스가 남쪽(South) 방향으로 증가!
  - S면 = J 증가 방향
  - N면 = J 감소 방향

이전 코드 오류:
  j_nd = J-1 (0-based)로 변환 후, j+1 = N 방향으로 가정했으나
  실제로는 j+1 = J+1 = S 방향

영향:
  - 50:50 양수/음수 문제가 이 매핑 오류에서 기인할 가능성
  - Albedo 캘리브레이션 재수행 필요 (N/S 매핑 수정 후)
