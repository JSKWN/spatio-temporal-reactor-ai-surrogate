## 1. 설계 결정

### 1.1 원본 격자 분리 저장

σ₀(N1=72점)와 미분계수(N2=14점)의 BU 격자가 다름 → **사전 보간 하지 않고 원본 격자 그대로 저장**.

| 격자 | 변수 | 크기 | 값 | 용도 |
|------|------|:----:|-----|------|
| 0B | N1 (BURN) | 72 | [0, 0.1, 0.5, 1.0, ... 60.0] MWD/kgHM | σ₀ base XS |
| 1B | N2 (VAR) | 14 | [0, 0.5, 1.0, 3.0, 5.0, 7.0, 10, 15, 20, 25, 30, 40, 50, 60] | 미분계수 |
| ΔM | N3 (DMOD) | 6 | [+0.301, +0.247, +0.171, +0.093, +0.042, -0.054] g/cc | 밀도 piecewise |

### 1.2 밀도 piecewise: 6구간 전부 저장

`d_sigma_dM`을 (Z, qH, qW, 2, **6**)으로 저장. 런타임에 현재 밀도로 보간.

- dmod_delta = [+0.301, +0.247, +0.171, +0.093, +0.042, -0.054] g/cc
- 실제 밀도 = REFDM + ΔM → [1.005, 0.950, 0.875, 0.797, 0.746, 0.650] g/cc
- 운전 밀도 ~0.66 → j=6 근방

## 4. Phase F 확장 구조 (multi-BU 테이블)

```
fixed/
├── xs_fuel                      (72, Z, qH, qW, 10)   float32  ← BU 차원 추가
├── xs_fuel_bu_grid              (72,)                  float64
│
├── taylor_xe35/
│   ├── ref_conditions           (4,)                   float64
│   ├── bu_grid_base             (72,)                  float64  ← σ₀ BU 격자
│   ├── bu_grid_deriv            (14,)                  float64  ← 미분계수 BU 격자
│   ├── dmod_delta               (6,)                   float64
│   │
│   ├── sigma_0                  (72, Z, qH, qW, 2)    float64  ← N1 격자
│   ├── d_sigma_dP               (14, Z, qH, qW, 2)    float64  ← N2 격자
│   ├── d_sigma_dTf              (14, Z, qH, qW, 2)    float64  ← N2 격자
│   └── d_sigma_dM               (14, 6, Z, qH, qW, 2) float64  ← N2 × N3
│
├── taylor_sm49/                                        ← SM49 (동일 구조)
├── taylor_sb10/                                        ← SB10
└── ...
```

### 런타임 보간 전략

```python
# 1. BU 보간 — σ₀와 미분계수 각각 보간
sigma_0    = interp1d(bu_grid_base,  sigma_0_table,  axis=0)(Bu_current)   # (Z,qH,qW,2)
d_sigma_dTf= interp1d(bu_grid_deriv, d_dTf_table,    axis=0)(Bu_current)   # (Z,qH,qW,2)

# 2. 밀도 보간 — 현재 밀도에서 적절한 j 보간
actual_densities = REFDM + dmod_delta  # (6,)
d_sigma_dM_at_rho = interp1d(actual_densities, d_dM_table, axis=1)(rho_current)

# 3. Taylor 전개
sigma = sigma_0 + d_sigma_dTf * Δ√Tf + d_sigma_dM_at_rho * Δρ
```

### BOC 런타임 (BU=0, BU 보간 불필요)

```python
sigma_0    = h5['fixed/taylor_xe35/sigma_0'][:]        # (Z,qH,qW,2)
d_dTf      = h5['fixed/taylor_xe35/d_sigma_dTf'][:]    # (Z,qH,qW,2)
d_dM_all   = h5['fixed/taylor_xe35/d_sigma_dM'][:]     # (Z,qH,qW,2,6)
dmod_delta = h5['fixed/taylor_xe35/dmod_delta'][:]      # (6,)
ref        = h5['fixed/taylor_xe35/ref_conditions'][:]  # (4,)

# 밀도 보간
actual_densities = ref[3] + dmod_delta  # [1.005, ..., 0.650]
# 현재 ρ_m=0.657 → j=6 근방 보간

# Taylor 전개
sigma = sigma_0 + d_dTf * (√(Tf+273.15) - √(ref[1]+273.15)) + j_interp * (ρ_m - ref[3])
```

## 5. 공간 매핑 흐름

```
1. xsl_taylor_parser.parse_taylor_xe35(xsl_path)
   → List[TaylorXe35] (16개 조성)

2. xs_voxel_builder._build_fuel_string_voxel(geom)
   → string_voxel (Z, 9, 9) — 각 셀 = 조성명

3. quarter crop → (Z, qH, qW)

4. 조성명 → Taylor 계수 매핑 (BOC, bu_index=0):
   string_voxel[z, y, x] = "PPE_272_808"
                      ↓
   sigma_0[z, y, x, :]       = [7.267, 1172690]           (2,)   [barn]
   d_sigma_dP[z, y, x, :]    = [1.20E-4, -50.9]           (2,)   [barn/ppm]
   d_sigma_dTf[z, y, x, :]   = [-8.20E-3, 654]            (2,)   [barn/√K]
   d_sigma_dM[z, y, x, :, :] = [[0.96, 1.04, ...], [...]] (2, 6) [barn/(g/cc)]

5. HDF5에 1회 저장
```

## 6. 검증 기준

1. HDF5 `sigma_0` 중앙 노드 값 = PPE_272_808의 σ₀ 정확 일치
2. Taylor σ 계산 → E3-2 결과 재현 (g=1: 0.25%, g=2: 0.92% vs ABS-XEN)
3. `ref_conditions` 4개 값 정확 복원
4. `d_sigma_dM` j=6 값 → 운전 밀도(~0.66) 근방 σ 계산 정확도 개선
5. `dmod_delta` = COMP 헤더 ΔM 일치
