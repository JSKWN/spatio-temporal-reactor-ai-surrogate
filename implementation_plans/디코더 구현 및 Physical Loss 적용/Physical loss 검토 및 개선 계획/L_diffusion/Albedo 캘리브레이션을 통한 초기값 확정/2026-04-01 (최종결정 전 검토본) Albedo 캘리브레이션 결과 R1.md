# Albedo 캘리브레이션 결과 R1 — 행렬 Albedo

## 목적
스칼라 α의 한계(g2 축방향 음수) 해결을 위한 2군 행렬 C/β 캘리브레이션.

## 용어 정리

### β (albedo matrix, 반사율 행렬)
반사체 경계에서 나간 중성자가 되돌아올 확률의 2×2 행렬:
```
β₁₁: 고속으로 나간 중성자 → 고속으로 돌아옴 (동일군 반사)
β₂₂: 열로 나간 중성자 → 열로 돌아옴 (동일군 반사)
β₂₁: 고속으로 나간 중성자 → 열로 돌아옴 (반사체 내 감속 후 반환) ← 핵심
β₁₂: 열로 나간 중성자 → 고속으로 돌아옴 (역감속, 물리적으로 ~0 기대)
```
값 범위: 0(흡수) ~ 1(완전 반사)

### C (coupling matrix, 커플링 행렬)
β로부터 유도된 실용적 형태: **J_net = C × φ**
```
C = ½(I+β)⁻¹(I-β)
```
- C가 크면 누설 큼, 작으면 누설 적음, 음수면 역류
- L_diffusion 경계면 loss에 직접 대입 가능

### R² (결정계수)
J_net = C×φ 모델이 데이터를 얼마나 잘 설명하는지:
- R² = 1.0: 완벽 (C×φ로 J_net을 정확히 예측)
- R² = 0.0: 전혀 설명 못함
- R² < 0.1: 선형 모델 부적합 → 다른 접근 필요

### 독립변수 충분성
J_net = C × φ에서 독립변수는 φ₁, φ₂ 두 개.
확산 이론 BC에서 경계면 net current는 경계면 flux의 선형 함수이므로,
**반사체 물성(재질, 두께)이 일정하면 C는 상수 → φ₁, φ₂만으로 충분**.
(온도/밀도 의존성은 현재 제외)

## 역산 방법
```
least squares: [J1(s)]   [C11 C12] [φ1(s)]
               [J2(s)] = [C21 C22] [φ2(s)]
→ 행별 독립 최소자승 회귀
```

## 결과

### Bottom (Z 하부, 물 반사체) — R² 우수
```
C = [+0.156  -0.142]    β = [+0.538  +0.378]
    [-0.025  +0.079]        [+0.067  +0.744]
R²: g1=0.999, g2=0.990
|C₂₁|/|C₂₂| = 32.0% → C₂₁ 유의미 → 행렬 BC 필요
```
- C₂₁ = -0.025 < 0: φ₁(고속) 증가 시 g2 누설 감소(=열중성자 유입 증가)
  → 물 반사체에서 고속→열 감속 후 반환 효과

### Top (Z 상부, 물 반사체) — R² 우수
```
C = [+0.144  +0.027]    β = [+0.550  -0.072]
    [-0.035  +0.076]        [+0.095  +0.731]
R²: g1=0.995, g2=0.914
|C₂₁|/|C₂₂| = 46.4% → C₂₁ 유의미 → 행렬 BC 필요
```
- C₂₁이 bottom보다 더 큼 (46% vs 32%) → 상부 반사체 감속 반환 효과 더 두드러짐

### Radial (XY 측면, SS+H₂O) — R² 매우 낮음
```
C = [+0.032  -0.144]    β = [+0.878  +0.515]
    [+0.000  +0.027]        [-0.001  +0.899]
R²: g1=0.001, g2=0.108
|C₂₁|/|C₂₂| = 0.7% → C₂₁ 무시 가능
```
**문제**: R²가 매우 낮음 → J_net ≈ C×φ 선형 모델이 방사방향에서 부적합.
원인 후보: 경계면 식별 로직 오류 (양수/음수 50:50 혼재), 위치별 반사체 구성 차이.
별도 분석 필요.

## 핵심 판단

| 방향 | C₂₁/C₂₂ | R² | 판정 |
|------|:---:|:---:|------|
| radial | 0.7% | 0.001~0.108 | R² 문제 → 별도 분석 필요 |
