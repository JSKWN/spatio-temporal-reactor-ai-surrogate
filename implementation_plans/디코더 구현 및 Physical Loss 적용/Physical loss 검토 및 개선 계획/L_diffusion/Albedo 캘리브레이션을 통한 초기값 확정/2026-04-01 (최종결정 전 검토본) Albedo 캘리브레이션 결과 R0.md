# Albedo 캘리브레이션 결과 (R0 — 초기 시도)

## 목적
L_diffusion 반사체 경계 처리를 위한 학습 가능 Albedo α의 **데이터 기반 초기값** 도출.

## Albedo의 물리적 의미

### 원래 경계조건: 반사율 β
반사체 경계에서 나가는 partial current j⁺의 일부가 되돌아옴:
```
j⁻ = β × j⁺      (β = albedo = 반사율)
```
- β = 0: 진공 (반사 없음), β = 1: 완전 반사체

### α와 β의 관계 (Marshak 변환)
```
α = (1 - β) / (1 + β)
```
- β↑(반사↑) → α↓ → 누설↓
- β↓(반사↓) → α↑ → 누설↑

### Marshak BC에서 α의 역할
net current를 경계면 flux로 표현:
```
J_net = α/2 × φ_boundary
```
FD 형태 (φ_boundary → 노드 중심 φ̄):
```
J_net = αD/(αh/2 + D) × φ̄
```
**α가 클수록 누설이 크고, 작을수록 누설이 적음.**

### 엄밀 형태: Albedo Matrix (2군)
스칼라가 아닌 2×2 행렬:
```
j_g⁻ = Σ_{g'} β_{gg'} × j_{g'}⁺

  [ j₁⁻ ]   [ β₁₁  β₁₂ ] [ j₁⁺ ]
  [ j₂⁻ ] = [ β₂₁  β₂₂ ] [ j₂⁺ ]
```
- β₁₁: 고속→고속 반사 (대각)
- β₂₂: 열→열 반사 (대각)
- β₂₁: 고속→열 전달 (비대각) — 반사체 내 감속 후 열중성자로 반환

현재 per-group scalar α는 **대각 성분만 취한 근사**.
비대각 성분(cross-group 기여)은 g2의 α에 암묵적으로 합산되는 효과.

## 역산 수식

```
Marshak FD: J = αD/(αh/2 + D) × φ̄    (per-unit-area)
GT 경계 누설: gt = 2 × JNET0_reflector_face
역산: α = gt×D / (D×φ̄ - gt×h/2)
```

h: radial = 10.804 cm, axial = 10 cm

## 물리적 해석

### Fast Group (g1): α > 0 — 중반사체 효과
- V-SMR(SMART) 반사체: SS baffle/barrel + 경수 → 중반사체(Heavy Reflector)
- SS의 높은 원자번호 밀도 → 고속중성자 탄성산란(반사) 지배
- α < 0.4692(이론값) → β(반사율) 큼 → 물리적으로 타당

### Thermal Group (g2) 축방향: α < 0 — Thermal Flux Peaking
- **물 반사체(축방향)에서 발생하는 물리 현상**:
  1. 고속중성자가 물 반사체로 누설
  2. 물에 의해 감속 → 열중성자로 변환
  3. 반사체의 열중성자 flux > 노심 → **열중성자 역류** (J_net < 0)
- bottom_g2: **100% 음수** → median = -0.017
- top_g2: **100% 음수** → median = -0.041
- **스칼라 α의 한계**: albedo matrix의 비대각 성분 β₂₁(고속→열 전달)을
  스칼라 1개로 표현 불가

### Thermal Group (g2) 방사방향: α > 0
- SS(Fe)가 열중성자를 강하게 흡수 → 진공 대비 누설 증가 → α > 0.4692

### 방사방향 양수/음수 50:50 혼재
- radial에서 양수:음수 = 14400:14400 (정확히 50:50)
- 추후 확인 필요: 대칭면 vs 실제 반사체면 식별 로직
