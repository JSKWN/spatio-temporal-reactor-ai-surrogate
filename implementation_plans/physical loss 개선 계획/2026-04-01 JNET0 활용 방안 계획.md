## 작업 A: 반사체면 Albedo 최적값 역산 (논문 근거자료 포함)

**목적**: 학습 가능 α의 초기값을 이론값(0.4692) 대신 **데이터 기반 최적값**으로 설정.
추후 논문 작성 시 "왜 이 초기값을 선택했는가"에 대한 정량적 근거 확보.

**방법**:
반사체 인접 연료 노드에서, JNET0×2(GT neutron current)와 Marshak BC 수식을 대조:
```
GT_leak_boundary = 2 × JNET0_reflector_face × A_face
Marshak: leak = α × D/(D + α/2) × φ̄ × A_face
→ α_optimal 역산
```

**세분화**: 방향 3종 × 군 2개 = 6개 α 각각 역산
- α_radial_g1, α_radial_g2 (XY 측면 반사체면)
- α_bottom_g1, α_bottom_g2 (Z 하부 반사체면, K=2의 B면)
- α_top_g1, α_top_g2 (Z 상부 반사체면, K=21의 T면)

**통계**: LP 2개 × CRS 10스텝 × 경계 노드 수 → 각 α의 median, std, min, max

## 작업 B: D̂ 보정항 역산

NNEM에서 CMFD neutron current와 NEM neutron current의 차이가 D̂:
```
J_NEM = D̃(φ̄₂ - φ̄₁)/h + D̂(φ̄₂ + φ̄₁)
```
JNET0×2 = J_NEM (MASTER GT), CMFD = D̃(φ̄₂ - φ̄₁)/h (우리 계산)이므로:
```
D̂ = (JNET0×2 - CMFD_J) / (φ̄₂ + φ̄₁)
```

**활용**: D̂ 분포의 통계적 특성 분석 → 학습 가능 보정항 설계 가능성 검토
- D̂가 위치/조성에 따라 체계적 패턴 → 보정 모델 학습 가능
- D̂가 무작위 → CMFD 한계 인정, 보정 불필요

**산출물**:
- 테스트 스크립트: `piecewise-test/2026-04-01_dhat_analysis.py`
- 실행 결과: `piecewise-test/2026-04-01_dhat_analysis_output.txt`

## 작업 C: L_diffusion irreducible floor 설정

JNET0×2(밸런스 ~0%)와 CMFD 밸런스(잔차 2~7%)의 차이 = L_diffusion의 달성 불가능 하한.
→ 훈련 시 loss가 이 floor 이하로 내려가면 과적합 의심 → 조기 종료/가중치 감소 기준

이 값은 이미 §3.4.3 (노드 vs 집합체 비교)에서 확인됨:
- 집합체 CMFD: g1 ~2.3%, g2 ~7.5%
→ 추가 계산 불필요, 레퍼런스 §3.4.5에 이미 기술
