# Albedo 캘리브레이션 결과 R3 — JNET0 부호 확정 + 최종 Albedo

## JNET0 부호 규약 확정

대칭 노드 쌍(4중 대칭 노심)에서 raw JNET0 비교:
```
N-boundary (19,10): JNET0-N = -1.950e+12
S-boundary ( 2,11): JNET0-S = -1.950e+12  ← 부호 동일, 크기 동일

E-boundary (10,19): JNET0-E = +2.387e+12
W-boundary (11, 2): JNET0-W = +2.387e+12  ← 부호 동일, 크기 동일
```

**결론: JNET0은 외향(outward) 방향 규약**.
절대 좌표(+X, +Y) 방향이 아님. 부호 반전 불필요.

## 50:50 양수/음수 분할의 원인 — 물리 현상

반사체 인접 연료 노드의 JNET0이 **음수인 면이 존재**:
- JNET0 < 0: 반사체에서 노심으로 중성자 **유입** (반사체의 반사/산란 효과)
- JNET0 > 0: 노심에서 반사체로 중성자 **누설**

N-boundary node의 JNET0-N = **-1.95e12** → 이 면에서는 중성자가 노심으로 들어옴.
이는 부호 규약 버그가 **아니라**, V-SMR의 두꺼운 중반사체(SS+H₂O)가 고속중성자를
효과적으로 반사하여 일부 면에서 순유입이 발생하는 물리적 현상.

## Per-face Marshak α 역산 (모델 C 기반, 외향 규약)

**방법**:
1. 모델 C: J_refl = f(φ₁, φ₂, JNET_opp_g1, JNET_opp_g2) → R²>0.97
   → 각 면의 J_refl을 정확히 예측
2. Per-face Marshak: α = J_refl × D / (D × φ - J_refl × h/2)
   → 각 면의 D(확산계수)를 개별 사용

**J = C×φ 선형 모델의 R²가 낮았던 이유 (테스트 1,2)**:
Marshak FD: J = α×D/(αh/2+D)×φ 에서 **D가 노드마다 다름** (연료 조성 차이).
같은 α라도 D가 다르면 J/φ가 달라짐.
J = C×φ는 모든 면에 단일 C → D 변동이 잔차 → R²≈0.
Per-face Marshak은 각 면의 D를 개별 사용하므로 이 문제 해소.

**결과**:

| 그룹 | 군 | 양수 α (median ± std) | β = (1-α)/(1+α) | 양수:음수 |
|------|:--:|:---:|:---:|:---:|
| ortho | g1 | 0.100 ± 0.015 | 0.818 | 4800:4800 |
| ortho | g2 | 0.400 ± 0.148 | 0.429 | ~50:50 |
| diag | g1 | 0.081 ± 0.021 | 0.850 | 9600:9600 |
| diag | g2 | 0.549 ± 0.295 | 0.291 | ~43:57 |

- g1: std 작음 (15~26%) → ortho/diag 내에서 albedo **일정** ✓
- g2: std 상대적으로 큼 → 열중성자 흡수의 에너지 민감도 반영
- 50:50: 부호 버그 아님. 물리적으로 유입면/누설면이 공존.

**물리적 해석**:
- 고속(g1): SS 탄성산란 → 강한 반사 → β=0.82~0.85, α=0.08~0.10
- 열(g2): SS 흡수 → 약한 반사 → β=0.29~0.43, α=0.40~0.55
- ortho(SS↑) vs diag(H₂O↑): 열중성자 기준 ortho가 더 반사(β 0.43>0.29)
  → SS가 물보다 열중성자 반사에 기여 (물은 감속+흡수)

## 최종 초기 Albedo 값

### Radial (per-face Marshak 기반, 양수 α median)

## Fick's law 검증 (MAS_NXS 직접 검증)

JNET0 < 0 면에서 반사체 φ가 연료 φ보다 높은지 직접 확인:

```
JNET0 > 0 (누설면, 36면): φ_fuel > φ_refl → Fick 일치 36/36 (100%) ✓
JNET0 < 0 (유입면, 36면): φ_refl > φ_fuel → Fick 일치  0/36 (  0%) ✗
```

**JNET0 < 0 면에서도 연료 φ > 반사체 φ** → Fick's law(J ∝ φ̄_high→φ̄_low) 불일치.

**의미**: "반사체에서 유입"이 아님. **NEM 고차 flux 전개(a₁, a₂) 효과**로,
특정 면의 surface gradient가 node-average gradient와 반대.
→ 노드 평균 flux만으로는 면 전류 방향을 예측 불가
→ 이전 분석(J_net ≠ f(φ̄), 반대면 JNET0 추가 시 R²=0.99)과 완전히 일관

## g2(열중성자) 검증 — g1과 동일 패턴 확인

S-boundary 노드 (j=2, i=10), K=11:
```
g1: JNET0_S = -1.95e+12  φ_fuel=5.95e+13 >> φ_refl=1.26e+13  Fick ✗
g2: JNET0_S = -1.06e+11  φ_fuel=9.42e+12 >> φ_refl=2.22e+12  Fick ✗
```

g1/g2 모두: φ_fuel >> φ_refl인데 JNET0 < 0 (유입) → **NEM 고차 효과가 군에 무관하게 일관 발생**.
g1이 g2보다 |JNET0| 10배 큼 → 고속중성자의 반사체 반사 효과가 더 강함.
